# Random Users App 📱.

Flutter app that fetchs data with the help of [RandomUser](https:www.randomuser.me) API.

⭐️ this repo if you like it.

## Getting Started 🚀

- Clone the repo
- Install the dependicies
- Run it

```flutter
flutter run
```
